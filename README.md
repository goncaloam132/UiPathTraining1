# UiPath Email Automation

## Descrição
Automação que lê emails e guarda ficheiros anexados numa pasta.

## Tecnologias
- UiPath Studio
- Gmail Activities

## Funcionalidades
- Ler emails da Inbox
- Criar pasta "Invoices" para guardar os ficheiros, neste caso faturas.
- Processar emails automaticamente

## Segurança
Credenciais geridas via UiPath (não incluídas no repositório)

## Como executar
1. Abrir no UiPath Studio
2. Configurar ligação ao Gmail
3. Executar Main.xaml
